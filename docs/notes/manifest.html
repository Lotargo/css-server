Runtime manifest

DOM as memory. CSS as processor. JavaScript as transport.

CSS-Server treats DOM nodes as visible task cells. CSS custom properties, typed attributes, selectors, and `calc()` resolve local computation. JavaScript is intentionally demoted to orchestration, I/O, and observation.

The calculator proved deterministic local behavior. The GitHub Pages site reframes that proof as a static runtime that can later attach outbound network transport.
