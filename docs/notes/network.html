Network Lab boundary

Fetch is transport, not the computation core.

JavaScript owns browser-only I/O: collecting endpoint fields, calling `fetch`, reading response text, and writing normalized metadata into DOM attributes.

CSS owns visual classification where practical. The current Network Lab classifies HTTP status families from `data-status` selectors and derives latency/payload/body/content badges from raw DOM attributes.

JavaScript writes raw transport facts such as status, elapsed milliseconds, payload length, and response text. It does not hide provider SDK behavior or ship shared API keys.
