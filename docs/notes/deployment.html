Deployment boundary

GitHub Pages serves static files only.

CSS-Server does not claim that GitHub runs backend code. The static site is delivered by Pages, then the browser runs the DOM/CSS computation substrate locally.

Allowed: browser-local computation, user-triggered outbound Web APIs, and transparent runtime diagnostics. Not allowed: bundled shared secrets, hidden traffic, inbound server claims, mining, spam, scraping at scale, or abusive automation.
